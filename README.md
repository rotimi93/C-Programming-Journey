# C-Programming-Journey
C++ Learning Journey – From Basics to ATM Simulation
-------------------------
This repository contains my C++ practice projects from my first day learning the language to building a functional ATM banking system simulation.

The goal of this repository is to document my learning progression, problem-solving approach, and increasing understanding of core C++ concepts.
-------------------------
Projects Included:

1. Pointer and memory address practice
2. Function and reference parameter exercises
3. Student name and score collection system
4. Multi-user ATM banking system simulation
--------------------------
Concepts Covered:

- Variables and data types
- Arrays
- Functions
- References and parameter passing
- Loops (for / while)
- Conditional logic (if / switch)
- Basic authentication logic
